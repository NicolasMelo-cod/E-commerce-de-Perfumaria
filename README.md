Loja de Perfumaria

🛍️ E-commerce de Perfumaria

Projeto desenvolvido em equipe com o objetivo de simular uma aplicação real de e-commerce, envolvendo integração entre frontend e API.

👨‍💻 Minha contribuição

Atuei diretamente no desenvolvimento do frontend da aplicação e na integração com a API, sendo responsável por:

Construção das interfaces utilizando Angular
Implementação da navegação entre páginas
Consumo e integração com endpoints da API
Organização da estrutura do projeto no frontend
Correção de bugs e resolução de conflitos de código (Git)

🚀 Tecnologias utilizadas
Angular
TypeScript
HTML
CSS

💻 Funcionalidades
Listagem de produtos
Visualização de detalhes dos produtos
Área administrativa (CRUD de produtos)
Navegação entre páginas
Integração com API
